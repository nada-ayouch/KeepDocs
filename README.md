# KeepDocs

## Overview

KeepDocs is a web-based document management and archiving system.

The application allows administrators and employees to manage documents efficiently through a simple and user-friendly interface.

## Features

* User authentication and session management
* Employee management
* Department/Service management
* Document upload and archiving
* Profile picture management
* Search and filtering
* Pagination
* Role management (Admin and Employee)
* Document download and deletion

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* XAMPP

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/KeepDocs.git
   ```

2. Place the project in the `htdocs` folder of XAMPP.

3. Create a MySQL database named `keepdocs`.

4. Import the SQL database file.

5. Configure the database connection in `dbconnect.php`.

6. Start Apache and MySQL from XAMPP.

7. Open the project in your browser:

   ```
   http://localhost/KeepDocs
   ```
